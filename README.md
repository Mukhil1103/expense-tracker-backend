# Expense Tracker Backend

Django REST API with JWT authentication for managing expenses.

## Features
- JWT Authentication (Login / Refresh)
- CRUD operations for expenses
- Protected endpoints
- RESTful API design

## Tech Stack
- Django
- Django REST Framework
- SimpleJWT

## API Endpoints

- /api/token/ → Login
- /api/expenses/ → CRUD operations
- /api/categories/

## Deployment
Hosted on Render
